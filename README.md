# 회지 메이커 ❦

AI 챗 **롤플레이 로그**를 붙여넣으면 **소설책/회지 느낌**으로 정리해 주고,
**PDF로 저장**할 수 있는 웹앱이에요. 백엔드 없이 브라우저에서만 동작합니다.

## 입력 규칙

| 입력 | 인식 | 책에서 보이는 모습 |
| --- | --- | --- |
| `*그는 문을 열었다*` | 행동지문(나레이션) | 들여쓴 본문 문장 |
| `"늦어서 미안"` | 대사 | “늦어서 미안” |
| `늦어서 미안` (그냥 텍스트) | 대사 | “늦어서 미안” (자동 따옴표) |
| 빈 줄 2개 이상 또는 `***`, `---` | 장면 구분 | ❖ ❖ ❖ |

한 줄에 지문과 대사를 섞어 써도 돼요. 예) `*그가 웃으며* 안녕`

## 기능

- 제목 / 부제(저자) → 표지 자동 생성
- 판형 선택: A5 · A4 · B6(문고본)
- 본문 글꼴(나눔명조 · 본명조 · 고운바탕 · 본고딕), 글씨 크기
- 지문 스타일(기울임 · 일반 · 흐리게), 드롭캡, 자동 따옴표 토글
- 실시간 미리보기
- **PDF로 저장** 버튼 (브라우저 인쇄 → "PDF로 저장", 한글 폰트가 그대로 벡터로 들어감)

## 사용법

### 그냥 열기
`index.html` 파일을 브라우저로 열면 끝.

### 로컬 서버로 보기 (권장)
```bash
python3 -m http.server 8000
# http://localhost:8000 접속
```

### 배포 (GitHub Pages)
저장소 Settings → Pages → Source를 `main` 브랜치 `/ (root)`로 지정하면
`https://<username>.github.io/<repo>/` 에서 바로 열려요.

## 구조
```
index.html   # 화면 구조
styles.css   # 책/회지 스타일 + 인쇄용 CSS
app.js       # 로그 파서 + 렌더링 + PDF 저장
```

## PDF 팁
인쇄 대화창에서 **대상(저장 위치)을 "PDF로 저장"** 으로 선택하세요.
- 여백/배경 그래픽은 자동으로 인쇄에 맞게 정리됩니다.
- 표지 다음 페이지부터 본문이 이어집니다.
